# PedidosYa Dashboard in Tableau

This project showcases an interactive **Tableau dashboard** built using real delivery data from **PedidosYa Argentina**. The goal is to clearly and efficiently visualize key business insights, including:

- Orders by Geographic Area
- Total Revenue by City and Food Category 
- Total Orders
- Total Express Orders 
- Payment Method
- Orders by City and Delivery Method
- Rating by Food Type
