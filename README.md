# PedidosYa Dashboard en Tableau

Este proyecto muestra un tablero interactivo creado en **Tableau** con datos de la plataforma **PedidosYa Argentina**. El objetivo es visualizar de manera clara y eficiente el comportamiento de los pedidos, incluyendo métricas clave como:

- Pedidos por zona geográfica
- Ganancias por zona según categoría de comida
- Pedidos totales
- Pedidos express
- Formas de pago
- Pedidos por ciudad y método de entrega
- Puntaje por tipo de comida
